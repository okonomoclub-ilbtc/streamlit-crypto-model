# streamlit-crypto-model
 La aplicación muestra datos históricos recientes, una tabla detallada de precios previstos para los días futuros (hasta 15 días), incluyendo una columna "Tendencia" que indica "ALZA" o "BAJA" en relación con el precio del día anterior, y un gráfico combinado que visualiza tanto los precios históricos como los de predicción.
